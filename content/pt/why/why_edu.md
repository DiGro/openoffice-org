![Porquê o Apache OpenOffice: escolas](/why/images/why_edu.png)

Estabelecimentos de ensino de todos os níveis (primário, secundário, faculdade, universidade ...) afirmam que o Apache OpenOffice atende às necessidades de 
professores e alunos. O **processador de texto flexível**, a **poderosa folha de cálculo**, **gráficos dinâmicos**, **acesso a bases de dados** e mais, 
cumprem todos os requisitos para um conjunto de programas de escritório.

Com uma **licença de código aberto (open-source)**, o OpenOffice pode ser livremente usado e distribuído sem preocupações de licenciamento.

  - **Para os alunos e estudantes**
  
    O Apache OpenOffice forma uma plataforma de ensino ideal para competências nucleares de literacia informática, sem amarrar os alunos a produtos comerciais. 
	A licença de programa grátis significa que os alunos podem receber cópias do programa para usar em casa - de forma perfeitamente legal - um "valor 
	acrescentado" útil. Para os estudantes de Tecnologias de Informação, o OpenOffice, sendo baseado em componentes, também é uma plataforma ideal para 
	o desenvolvimento de competências em Tecnologias de Informação e entender a engenharia de programação real.

  - **Para os professores e académicos**

    O Apache OpenOffice também é uma plataforma ideal para a criação de materiais de ensino e gestão de tarefas administrativas. Por exemplo, o *Writer*, 
	processador de texto, é fácil de usar para notas simples mas poderoso o suficiente para lidar com teses complexas. Para o pessoal de Tecnologias de Informação, 
	a licença de programa de código aberto significa o fim das preocupações de conformidade da licença e da ameaça de auditorias ao programa. O OpenOffice 
	é desenvolvido, traduzido e apoiado por uma comunidade internacional ligada pela Internet, abrindo possibilidades interessantes para projectos da
	escola.

	- ** Aberto para todos **
      O Apache OpenOffice é uma força líder internacional no movimento de inclusão digital - tornando programas da mais alta qualidade disponíveis 
	  para todos, independentemente dos rendimentos. O OpenOffice está disponível numa grande variedade de idiomas, e encorajamos activamente a 
	  produção de versões para os idiomas locais. Desenvolvemos o programa num processo de código aberto - o equivalente a publicação revista por utilizadores - criando um programa de alta qualidade.
