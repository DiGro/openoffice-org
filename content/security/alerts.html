Apache OpenOffice Security Alerts

The Apache OpenOffice Security Team publishes details of security vulnerabilities in our Security Bulletin.

We also publish these alerts via the project's announcement list, announce.

If you would like to subscribe to the list, please send a blank email to announce-subscribe@openoffice.apache.org. You will be sent an email from instructions how to confirm your subscription. Once you have confirmed your subscription, you will receive any future emails from announce until you unsubscribe.

Please note that the mailing list is fully automated, so if you use spam-filtering software, please make sure it will accept emails from announce@openoffice.apache.org before you try and subscribe.
