Quali piattaforme sono supportate dalla Suite OpenOffice.org?

Dal sito OpenOffice puoi scaricare i file binari per le seguenti piattaforme:

  • Linux (x86)
  • Solaris (Sparc)
  • Solaris (Intel)
  • Windows (in formato .zip)
  • Windows (in formato .tar)

Esiste anche un progetto di porting verso MacOS X.
