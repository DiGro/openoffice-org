Selezionate il vostro programma:
  • TAR

Usando TAR

  1. Scaricate il file in una cartella.  (Prendete nota in quale cartella avete salvato il file.)
  2. Da una finestra console, individuate e posizionatevi nella cartella dove avete salvato il file
  3. Digitate TAR -zxf NomeFile.  Dove NomeFile è il nome del file che volete estrarre.
