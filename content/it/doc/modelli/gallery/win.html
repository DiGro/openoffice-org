Windows - File Zip e GZ


Selezionate il vostro programma:
  • 7-Zip
  • WinZip

Usando 7-zip

  1. Scaricate il file in una cartella.  (Prendete nota in quale cartella avete salvato il file.)
  2. Minimizzate la finestra del vostro browser.
  3. Fate doppio clic su Risorse del Computer.
  4. Individuate la cartella dove avete salvato il file.
  5. Fate un clic con il tasto destro del mouse sul file
  6. Selezionate Estrai in...
  7. Digitate la cartella dove desiderate salvare i file decompressi.
  • Potete prelevare la vostra copia di 7-zip qui.
