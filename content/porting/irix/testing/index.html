TESTING STATUS

In our last most stable build of OpenOffice.org (OpenOffice641 / OO641B), we ran many test suites that exist in the OpenOffice.org tree. We obtained mixed results from all these tests, some passing, some failing, and others not even compiling.

As the OOO_STABLE_1 branch is no where near as stable as the 641B branch on IRIX, we thought it was time to revisit those tests once again, to check for any regressions.

The tests are currently a work in progress, so it may be a little while before the table in the OpenOffice 1.x.x section is complete.
