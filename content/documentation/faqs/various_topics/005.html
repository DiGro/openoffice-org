What hardware resources does OpenOffice need to work?

OpenOffice.org needs less resources than StarOffice. I tested it on a Pentium 133 MHz 32 MB RAM Windows 98 SE system and it worked fine. However, 64 MB RAM are recommended for better performances. Linux systems with a heavy windows manager or desktop environment may need more memory (64 MB RAM).

For developer specifications broken down by OpenOffice.org versions and the various operating systems, see the General System Requirements page.
