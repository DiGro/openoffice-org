Is it possible to install OpenOffice.org without the user having to input data?

Yes. For Openoffice.org1.1.x, the How To Install OOo with a Response File document explains how to do it.

For 2.x, the process is covered in Setup Guides for OpenOffice.org 2.x has some information that should help adapt the native installer process to your needs.
