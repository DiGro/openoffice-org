Ho scaricato il file binario di OpenOffice.org, ma quando provo ad aprirlo, mi viene chiesto di selezionare l'applicazione per farlo. Come posso aprire questo file?

Questo file potrebbe essere compresso. Decomprimi il file con un programma come WinZip, Pkzip o altri) o usa il comando tar (su sistemi Linux), poi riprova.
