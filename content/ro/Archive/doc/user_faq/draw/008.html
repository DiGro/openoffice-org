Cum pot salva un document Draw într-un format standard (BMP, JPG,...)?

Formatul .sxd folosit în Draw nu este un format grafic ci unul pentru documente.

Pentru a salva un document Draw într-un format grafic:

  1. Alegeţi: File-Export din bara de meniuri

  2. Alegeţi formatul grafic din caseta combinată 'File Type'

  3. Apăsaţi butonul 'Save'
