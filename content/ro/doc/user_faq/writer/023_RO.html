023 â Încerc să folosesc opţiunea de control al versiunilor pentru documentul meu dar anumite opţiuni din meniu pe care le-am văzut anterior nu mai apar. Ce este în neregulă?

Anumite funcţii de management al versiunilor devin absente când se crează un index (ca de exemplu un Cuprins) în document. Opţiunile de meniu care nu vor mai apărea:

  • Edit -> Changes -> Merge Document (Editare -> Modificări -> Contopire Document)

  • Edit -> Compare Document In addition (Editare -> Comparare Document în adăugire), butonul "Compare" din dialogul "Versions" (acesta folosind File -> Versions) devine gri (nefolosibil) când documentul conţine un index.
