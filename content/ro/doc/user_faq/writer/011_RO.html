011 - Cum pot crea etichete în OpenOffice.org?

Selectaţi: File -> New -> Labels. În fereastra de dialog apărută introduceţi textul iniţial pentru etichetă şi alegeţi tipul acesteia (Avery, etc.). Apăsaţi "New Document" pentru a deschide un nou document cu setările selectate.
