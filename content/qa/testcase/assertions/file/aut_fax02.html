Test Case: aut_fax02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create a Fax from Autopilot
Description:
    1. From the menu bar,  select File - Autopilot - Fax.
    2. The Autopilot Fax dialog should be launched.
    3. Ensure Modern is selected on the Autopilot Fax dialog.
    4. Select Create on the Autopilot Fax dialog.
    5. Ensure OpenOffice.org creates an Autopilot Fax document.
    6. The document should show
        A heading showing  "Fax Note"
        A heading showing "Number of pages incl. cover page:1"
        The start of the letter should begin with  "Dear Sir or Madam".
    7. Close the Fax Document.
     
     
Expected Result:

All operations should complete successfully.
 

End of test case
