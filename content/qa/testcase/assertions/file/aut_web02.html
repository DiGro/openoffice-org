Test Case: aut_web02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create an Autopilot web page
Description:
    1. From the menu bar,  select File - Autopilot - Web Page.
    2. The Autopilot Web Page dialog should be launched.
    3. Select Create on the Autopilot Web Page dialog.
    5. Ensure OpenOffice.org creates an Autopilot Web Page.

    7. Close the Web Page Document.

     
Expected Result:
All operations should complete successfully.
