Test Case: vew_bars04

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can toggle each toolbar on and off

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. From the menu bar, select   - view - toolbars,
        for each of the following toolbars toggle them off/on or on/off .
        View -Toolbars - Function Bar
        View -Toolbars - Object Bar
        View -Toolbars - Main Toolbar
        View -Toolbars - Formula bar
        View - Ruler - toggle off/on
        View - Status Bar - toggle off/on
    4. Close the text document.
Expected Result:
All operations should complete successfully.


End of test case
