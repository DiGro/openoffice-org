Test Case: spl_chk06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org spellcheck operates correctly.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. Type in the word 'score', then place cursor at the home position.
    3. From the menu bar select Tools - SpellCheck - Check
    4. The Spellcheck is complete prompt box should be displayed, click on OK.

    5. Type in the word 'sscore', then place cursor at the home position.
    6. From the menu bar select Tools - SpellCheck - Check
    7. The SpellCheck dialog box should appear and suggest 'score' as a replacement word.
    8. Select Replace. the word sscore should be replaced with score.
    9. The Spellcheck is complete prompt box should be displayed, click on OK.

    10 .Type in the word 'ssscore', then place cursor at the home position.
    11. Select Tools - SpellCheck - Check
    12. The SpellCheck dialog box should appear, there should be no suggested replacement word.
    13. Select Close. The word ssscore should be still in place.
    14. Close the text document.


     
Expected Result:
All operations should complete successfully.

End of test case
