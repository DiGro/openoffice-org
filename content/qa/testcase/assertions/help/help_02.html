Test Case: help_02

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org help options operate correctly.

Description:

    1. From the menu bar, select Help
    2. Ensure the Help Agent, Tips, and Extended tips can be toggled on/off.
    3. Select Help - About OpenOffice.org
    4. Ensure the OpenOffice.org splash screen is launched.
Expected Result:
All operations should complete successfully.

End of test case
