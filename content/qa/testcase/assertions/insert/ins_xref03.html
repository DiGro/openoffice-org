Test Case: ins_xref03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can launch the cross-reference fields dialog in a text document.

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. A new text document should be launched.
      3. From the menu bar,  select Insert - Cross-reference.
      4. The Fields dialog should be launched.
      5. Ensure there are 6 tab settings,
          Document, References, Functions, DocInformation, Variables and Database.
          Select each  tab setting.
      6. Select Close in the Fields dialog.
      7. Close the text document.
Expected Result:
All operations should complete successfully.

End of test case
