Test Case: ins_note03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can launch the insert note dialog correctly in a text document

Description:
 

1. From the menu bar,  select File - New - Text Document.
2. A new text document should be launched.
3. From the menu bar,  select Insert - Note.
4. The Insert Note dialog should be launched.
5. Click cancel on the Insert Note dialog.
6. Close the text document.
 
Expected Result:
All operations should complete successfully.

End of test case
