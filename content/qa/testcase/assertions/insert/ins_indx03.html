Test Case: ins_indx03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org launches the insert indexes dialog correctly in a text document.

Description:-

1. From the menu bar,  select File - New - Text Document.
2. From the menu bar,  select Insert - Indexes - Indexes & Tables.
3. The Insert Index/Table dialog should be launched.
4. Ensure there are 5 tab settings,
    Index/Table, Entries, Styles, Columns and Background.
    Select each tab setting.
5. Select Cancel in the  Index/Table dialog.
6. Close the text document.
Expected Result:
All operations should complete successfully.

End of test case
