QA API - Mission Statement

The OOo QA API Team has the main responsibility to assure and improve the quality of the OOo/SUN Staroffice Interface area. The testing happens mainly by using the office, sporadic functional testing, testing with TCS (Test Case Specification) and with automatic tests mainly on Windows, Linux and Solaris. Also new features are guided on their way into the product to assure their integration under best possible conditions.
