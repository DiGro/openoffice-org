OpenOffice.org Security

Übersicht über aktuelle Sicherheitsinformationen

Diese Seite befindet sich zurzeit im Aufbau. Für Fragen zu aktuellen Sicherheitsinformationen schreiben Sie bitte an presse@openoffice.org.

OpenOffice.org ist ein umfangreiches Softwarepaket, das von verschiedenen Teams entwickelt wird. Wie jedes große Softwarepaket kann es sicherheitsrelevante Fehler enthalten. Sollten Sie solche Fehler finden, kontaktieren Sie bitte das englischsprachige Security Team, sodass der Fehler untersucht und eine saubere Lösung für die Benutzer und alle folgenden Programmversionen erarbeitet werden kann.

Bitte geben Sie uns in der Fehlermeldung folgende Informationen:

  • In welcher Version von OpenOffice.org haben Sie das Problem entdeckt (z.B. 1.1.5, 2.0.2, etc.)?
  • Haben Sie eine offizielle Version von OpenOffice.org oder eine aus Ihrer GNU/Linux-Distribution?
  • Welcher Schaden entsteht (Datenverlust, Programmabsturz, Ausführung von Aktionen, etc.)?
  • Wie kann das Problem reproduziert werden?
  • Gibt es einen "Exploit", der diesen Fehler ausnutzt?
  • Wurde der Fehler bereits öffentlich gemeldet?

Nach Erhalt Ihrer Meldung werden wir den Fehler umgehend überprüfen und uns bei Ihnen melden.
